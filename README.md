# 🛠️ Vick's Store - Sistema de Inventario

Aplicación móvil desarrollada con Flutter para la gestión integral de inventario y monitoreo de estadísticas para una ferretería. 

Este proyecto forma parte de la clase de Desarrollo de Proyectos Full Stack del Instituto Tecnológico de Tepic.

## ✨ Características Principales

La aplicación cuenta con una interfaz moderna y fluida que incluye las siguientes pantallas funcionales:

* 🏠 **Inicio (Dashboard):** Resumen de estadísticas vitales, progreso del inventario y accesos rápidos a las funciones principales.
* 📷 **Escáner de Códigos:** Interfaz preparada para la captura de artículos mediante la cámara del dispositivo o entrada manual del código de barras.
* 📦 **Registro de Artículos:** Formulario estructurado y validado para añadir nuevos productos al catálogo con sus características físicas y de ubicación.
* 📋 **Inventario (Lista de Materiales):** Catálogo completo con tarjetas detalladas por producto, etiquetas de estado (Lleno, Medio, Vacío) y barra de búsqueda.
* 📊 **Reportes e Informes:** Panel visual con barras de progreso y estadísticas para analizar la distribución del estado del almacén.
* 👤 **Perfil de Usuario:** Pantalla de configuración de la cuenta y preferencias de la aplicación (Modo Oscuro).

## 💻 Tecnologías Utilizadas

* **Framework:** Flutter
* **Lenguaje:** Dart
* **Arquitectura:** Diseño modular separando componentes visuales (Widgets) y Vistas (Screens).
